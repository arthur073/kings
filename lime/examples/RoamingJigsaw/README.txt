--------------------
LimePuzzle README
--------------------

To build: javac *.java

To run, just load "LimePuzzleAgent" using the standard lime procedures.
Multiple agents will be able to collaborate while solving the puzzle.


Once you get started, the controls are:

right-click:       Selects a piece.
                   If you load a puzzle they are already all selected.

left-click-drag:   Moves a piece.
                   It must be selected with your color.

left-double-click: Rotates a piece.
                   You can also use the arrow keys or spacebar while dragging.
